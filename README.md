# Project KAVACH

## Cybersecurity Assessment Engagement

Project KAVACH is a simulated cybersecurity engagement focused on:

- Network Forensics
- Malware Traffic Analysis
- Web Application Security
- Threat Modeling
- Defense-in-Depth

---

## Objectives

- Analyze suspicious PCAP network traffic
- Identify malicious communication patterns
- Demonstrate OWASP vulnerabilities
- Propose remediation strategies
- Document findings professionally

---

## Technologies

- Ubuntu
- Wireshark
- Docker
- DVWA
- OWASP Juice Shop
- Burp Suite
- VS Code
- GitHub
